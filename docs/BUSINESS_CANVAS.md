# Sati Business Model Canvas

## 1. Customer Segments

- Primary: Knowledge workers in offices อายุ 25-45 ปี ที่นั่งหน้าจอ 6+ ชั่วโมงต่อวัน
- Secondary: บริษัทเทคและองค์กรที่ลงทุนด้าน employee wellness
- Tertiary: Co-working spaces, learning labs, gaming cafe, maker communities

## 2. Value Propositions

- Sensor-driven posture coach ที่ไม่ต้อง self-report
- Growth mechanic ต้นไม้ ทำให้อยากกลับมาดูทุกวัน
- HR aggregate view: บริษัทเห็นภาพรวม team wellness โดยไม่ละเมิด privacy รายคน
- เปิดเครื่อง เสียบ USB-C แล้วเริ่ม demo ได้เร็ว ไม่ต้อง setup ซับซ้อน
- Hardware ground truth ทำให้แตกต่างจากแอป wellness ที่ไม่มี sensor

## 3. Channels

- B2B: ขายตรงให้ HR, People Ops, Office Admin
- B2C: e-commerce, Arduino Store, maker community
- Community: Arduino Project Hub, wellness blogs, university innovation showcases
- Pilot program: ทดลองใช้ในบริษัท 2-4 สัปดาห์ พร้อม aggregate dashboard

## 4. Customer Relationships

- Self-service onboarding ผ่าน guided tour ในแอป
- Community Discord หรือ LINE OpenChat สำหรับ DIY tweak
- Enterprise: dedicated success manager สำหรับ pilot และ rollout
- Monthly insight report แบบ aggregate ให้ HR ใช้ปรับกิจกรรม wellness

## 5. Revenue Streams

- Hardware kit: UNO Q + Nano 33 BLE Sense + Modulino Distance แบบ one-time
- SaaS HR dashboard: per-seat monthly หรือ per-device monthly
- Premium themes และ decorative packs ในแอป
- Enterprise pilot package: setup + dashboard + workshop

## 6. Key Resources

- Arduino hardware ecosystem และ App Lab workflow
- Sensor fusion logic สำหรับ posture, distance, movement
- Web dashboard และ growth mechanic
- Content library สำหรับ stretch cue และ daily missions
- ทีม engineering ที่ทำ hardware + web integration ได้

## 7. Key Activities

- Hardware integration และ QA หน้างาน
- ปรับ threshold ให้เข้ากับสภาพโต๊ะและท่าติด sensor
- พัฒนา dashboard, mission, insight logic
- ทำ pilot กับทีม office จริงเพื่อเก็บ feedback
- สร้าง content สำหรับ guided wellness cues

## 8. Key Partners

- Arduino: hardware supply, App Lab ecosystem, community distribution
- Corporate wellness platforms สำหรับ integration ในอนาคต
- Ergonomics / workplace consultants สำหรับออกแบบ cue ที่เหมาะกับงานออฟฟิศ
- Coworking spaces หรือ university labs สำหรับ pilot

## 9. Cost Structure

- Hardware COGS: UNO Q, Nano 33 BLE Sense, Modulino, case, cable
- Engineering: firmware, Python bridge, web app, QA
- Cloud สำหรับ aggregate dashboard ในเวอร์ชันหลัง hackathon
- Customer support และ onboarding
- Content creation สำหรับ missions, stretches, themes

## Assumptions To Validate

- [ทีมต้องเติม] ราคาที่ลูกค้าองค์กรยอมจ่ายต่อ device / ต่อ seat
- [ทีมต้องเติม] BOM cost หลังเลือก enclosure และ mounting จริง
- [ทีมต้องเติม] Pilot conversion rate จากบริษัทเป้าหมาย
- [ทีมต้องเติม] Source สำหรับ market size ของ wellness tech หรือ corporate wellness ในไทย
